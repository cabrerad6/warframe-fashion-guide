HOW I MADE THIS

Project Process — CSIT 212 Final Project

Project Overview

This website — Warframe Fashion Guide (Fashionframe) — was built entirely through AI prompting using Claude (Anthropic) as the autocoding tool. No code was manually written or edited by the student. All HTML, CSS, and JavaScript was generated through a series of structured prompts, which are documented below.

The project demonstrates how modern AI tools can be used to design, build, and deploy a complete multi-page website from scratch using only natural language instructions.

AI Tool UsedClaude by Anthropic (claude.ai)
Deployed ViaGitHub Pages
Files Createdindex.html, gallery.html, generator.html, process.html, style.css, script.js, generator.js
Manual Code WrittenNone — 100% AI-generated

THE PROMPTS

Each prompt below was entered into Claude. The explanation describes what Claude produced as a result.

PROMPT 01
"Act as an AI autocoding web developer. I need to complete my CSIT 212 Final Project by building a public website completely through prompting, with no manual coding or deployment by me. My project idea is a website called 'Warframe Fashion Guide' that helps users find good Warframe fashion ideas. The website should give stylish Warframe looks with color palettes, armor/accessory suggestions, themes, and a gallery of builds. Help me build the full website with all files and give me step-by-step GitHub Pages deployment instructions."

What This Produced:

This was the master prompt that kicked off the entire project. Claude responded by generating the complete structure for a 4-page website: index.html (homepage), gallery.html (fashion gallery), generator.html (theme generator), and process.html (this page). It also produced style.css with a dark gaming aesthetic, script.js for navigation and interactivity, and generator.js with full fashion data for 6 themes. Additionally, Claude provided step-by-step GitHub Pages deployment instructions and future improvement prompts.

PROMPT 02
"Create a modern gaming-style website design for a Warframe fashion guide. Use a dark background with glowing accents, the Orbitron and Rajdhani fonts from Google Fonts, a CSS variable system for colors, and a sticky navigation bar. Make it feel like a premium game fan site."

What This Produced:

Claude generated the entire style.css file. This included a CSS variable system using :root with color tokens, a sticky navbar with a glowing logo, hero sections with animated grid backgrounds, card components with hover glow effects, and responsive layouts using CSS Grid and Flexbox. The Orbitron font is used for headings and the Rajdhani font for body text — giving the site a sci-fi gaming aesthetic.

PROMPT 03
"Build a Warframe fashion gallery page with 9 build cards. Each card should show a Warframe name, theme badge (dark, gold, neon, royal, angelic, infested), color swatches, and open a modal popup with full details including color palette with hex codes, armor recommendations, syandana suggestions, ephemera options, and style notes."

What This Produced:

Claude built the complete gallery.html page with 9 Warframe build cards including Excalibur Umbra, Rhino Prime, Volt Prime, Mag Prime, Wisp, Saryn Prime, Ash Prime, Nova Prime, and Mesa Prime. Each card has CSS-generated art, color swatches, and a clickable modal with full styling breakdowns. A filter bar lets users filter by theme. The JavaScript for filtering and modal open/close was included in script.js.

PROMPT 04
"Create an interactive Fashion Generator page where a user can: (1) click one of 6 theme buttons (dark, gold, neon, royal, angelic, infested), (2) then select from a list of recommended Warframes for that theme, and (3) receive a complete fashion blueprint with a color palette, armor picks, syandana, ephemera, and a style tip. Build this in JavaScript with all data hardcoded."

What This Produced:

Claude generated generator.html (the generator UI with a 3-step flow) and generator.js containing all the fashion data for every theme and Warframe combination. Clicking a theme reveals step 2 (Warframe selection), and choosing a Warframe generates a full result card with a color palette, themed lists, and a style tip. The system can also randomly regenerate a look using the "Regenerate Look" button.

PROMPT 05
"Add a homepage (index.html) that explains what the Warframe Fashion Guide does. Include a hero section with a title, subtitle, and two call-to-action buttons. Add a features section with 4 cards explaining what the site offers. Add a theme preview strip showing the 6 themes as clickable pills that link to the generator."

What This Produced:

Claude created the complete index.html homepage. The hero section has animated background grid lines, a large Orbitron title, a subtitle, and two CTA buttons. The features section uses a 4-column card grid with emoji icons. The theme preview strip shows all 6 themes as styled pill buttons that link directly to the generator page with a pre-selected theme parameter.

PROMPT 06
"Give me exact step-by-step instructions to deploy this website publicly on GitHub Pages with no experience required. Tell me exactly where to click, what to name the repository, and what the final public URL will look like."

What This Produced:

Claude provided a complete, beginner-friendly GitHub Pages deployment guide (reproduced in the Deployment section below). Instructions covered: creating a GitHub account, making a new repository, uploading files via the web interface, enabling GitHub Pages in repository settings, and the format of the final public URL. No terminal or command line was required.

PROMPT 07
"Make the website fully responsive for mobile. The navbar should collapse into a hamburger menu on small screens. The gallery grid should go to 1 column on phones. The generator theme cards should stack on mobile."

What This Produced:

Claude added CSS media queries to style.css for breakpoints at 768px and 480px. The navbar gained a hamburger icon (☰) that toggles a dropdown menu. Gallery and generator grids collapse to 1-2 columns on mobile. Font sizes scale down proportionally. The hero section padding adjusts for small screens so the title and buttons stay readable on phones.

🚀 GitHub Pages Deployment Guide

These are the exact steps used to publish this website publicly using GitHub Pages — no coding or terminal required.

1
Create a GitHub Account

Go to github.com and sign up for a free account if you don't have one.

2
Create a New Repository

Click the green New button. Name it exactly: warframe-fashion-guide. Set it to Public. Click Create repository.

3
Upload Your Files

In the new repository, click uploading an existing file. Drag and drop all 7 files: index.html, gallery.html, generator.html, process.html, style.css, script.js, generator.js. Click Commit changes.

4
Enable GitHub Pages

Go to your repository's Settings tab → scroll to Pages in the left sidebar → under Source, select Deploy from a branch → choose main branch → click Save.

5
Wait 1–2 Minutes, Then Visit Your Site

Your live URL will be: https://YOUR-USERNAME.github.io/warframe-fashion-guide/
Replace YOUR-USERNAME with your actual GitHub username.
